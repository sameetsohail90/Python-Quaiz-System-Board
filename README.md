# Smart Quiz System (Python Tkinter)

A desktop-based interactive **Smart Quiz System** built using Python and the Tkinter library. This application provides a clean graphical user interface (GUI) to conduct quizzes, manage multiple user attempts, and track scores seamlessly.

---

## 🚀 Features

*   **Interactive GUI:** Built with Python's Tkinter framework for a smooth user experience.
*   **Attempt Tracking:** Keeps track of multiple attempts made by users to monitor progress.
*   **Leaderboard System:** Features a leadership board to display top scores and encourage competition.
*   **Dynamic Scoring:** Instantly calculates and displays results at the end of the quiz.
